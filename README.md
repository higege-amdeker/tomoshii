# tomoshii

ろうそくの灯りをモチーフにした、静かに集中するためのシンプルなタイマーです。

## 概要

- 集中時間を 1 分単位で設定できます
- タイマーの進行に合わせて、ろうそくの演出が変化します
- HTML / CSS / JavaScript だけで動作します

## 公開

- Repository: https://github.com/higege-amdeker/tomoshii
- GitHub Pages: https://higege-amdeker.github.io/tomoshii/

## 使い方

1. 分数を入力するか、`+` / `-` ボタンで調整します。
2. スタートを押して集中を開始します。
3. 必要に応じてリセットで初期状態に戻せます。

## ローカルで確認する

`index.html` をブラウザで開くだけで動作します。

## ファイル構成

- `index.html`: 画面構造
- `style.css`: 見た目とアニメーション
- `script.js`: タイマー制御と画面更新
